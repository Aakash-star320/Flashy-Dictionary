# Flashy Dictionary

A lightweight Chrome extension that provides instant word definitions and audio pronunciations. Just double-click any word on a webpage to see its meaning!

## Features

- **Instant Definitions:** Double-click any word to view its definition in a non-intrusive popup.
- **Audio Pronunciation:** Listen to the correct pronunciation of the word directly from the popup.
- **Manual Search:** Use the extension popup to manually search for words.
- **Lightweight:** Built with vanilla JavaScript for fast performance.

## Installation

1.  Clone or download this repository.
2.  Open Chrome and go to `chrome://extensions/`.
3.  Enable **Developer mode** in the top right corner.
4.  Click **Load unpacked**.
5.  Select the folder containing this extension (`Myextension`).

## Usage

1.  **On Webpages:** Double-click on any word to see its meaning.
2.  **Popup:** Click the extension icon in the toolbar to open the search bar. Type a word and press "Go".

## Technologies Used

- JavaScript (ES6+)
- HTML5 & CSS3
- [Free Dictionary API](https://dictionaryapi.dev/)
